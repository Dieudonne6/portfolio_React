# 💼 Portfolio — Franck AYENAN

Personal portfolio built with **Next.js** and **React**, showcasing my background, skills, and projects as a backend developer.

---

## 🎯 Site Content

- **Hero / Home** with animated typewriter effect
- **About**
- **Skills**
- **Experience**
- **Education**
- **Certifications**
- **Projects**
- **Contact** (working form via EmailJS)
- Available in **French and English**

## 🛠 Tech Stack

| Technology | Role |
|---|---|
| Next.js 16 / React 19 | Front-end framework |
| TypeScript | Static typing |
| Tailwind CSS | Styling |
| shadcn/ui (Radix UI) | UI components |
| EmailJS | Contact form submission |
